*  Package Operations:
* =======================
*  This Package defines the Interface to the Code Publisher. The IExecutive is an abstract class that

*		- processcmdargs(int argc, char** argv)
*       - outputDir()
*       - ~IExecutive()
* These are inherited and overriden in the code publisher
